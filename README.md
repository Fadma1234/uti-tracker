# UTI Tracker

A simple web app to help track urinary tract infection (UTI) symptoms and monitor progress over time.  
Built with **HTML**, **CSS**, and **JavaScript**.

## Live Demo
Click the link on the right side of this page under **About** to open and try it out!

## Features
- Log daily UTI symptoms  
- Track changes over time  
- Simple and easy to use  
- Made to support awareness and self-monitoring
